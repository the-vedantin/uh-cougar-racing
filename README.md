# Cougar Racing static site

GitHub Pages mirror of `https://aayuyadav25.wixsite.com/mysite-1`.

This keeps the published Wix-rendered layout and CDN media so the visual output stays 1:1, with only the visible Wix ad/watermark removed and the 50px ad offset collapsed.

## Pages

- `index.html`
- `sub-team-page.html`
- `sponsors.html`
- `news.html`
- `inquiry-services-page.html`

## Notes

- Internal navigation is rewritten to the local `.html` files.
- `.nojekyll` is included for GitHub Pages.
- Media that is broken on the live Wix site is listed in `unavailable-media.txt`.
- The current file layout keeps every folder below 100 files.
